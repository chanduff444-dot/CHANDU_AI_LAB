# Security Policy

## Supported Versions

The following versions of **Chandu AI Lab** are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅ Yes     |
| 0.x     | ❌ No      |

---

## Reporting a Vulnerability

If you discover a security issue in **Chandu AI Lab**, please report it responsibly.

### 📩 How to Report

* Email: [chanduff444@gmail.com](mailto:chanduff444@gmail.com)
* Subject: **Security Vulnerability Report – Chandu AI Lab**

---

### 📝 What to Include

Please provide:

* Description of the vulnerability
* Steps to reproduce
* Possible impact
* Screenshots (if applicable)

---

### ⏱ Response Time

* Initial response: within **48–72 hours**
* Fix timeline: depends on severity

---

### 🔒 Responsible Disclosure

* Do NOT publicly disclose the issue before it is fixed
* Ethical reporting will be acknowledged

---

## Scope

This policy applies to:

* Local/offline application code
* AI processing modules
* Project scripts and automation

---

## Notes

Since this is an **offline AI system**:

* No cloud vulnerabilities are expected
* Most risks involve local misuse or code-level issues

---

Thank you for helping improve the security of **Chandu AI Lab** 🚀
